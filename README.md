# Sistema Distribuído de Criptomoeda

O projeto é um sistema distríbuido de criptomoeda simplificado e permissionado, em que os usuários podem criar carteiras, cosultar saldo, transferir moedas entre si e acompanhar histórico de transações.

Realizado como projeto para a disciplina ACH2147 - Desenvolvimento de Sistemas de Informação Distribuídos-, orientado pelo Prof. Renan Alves.

## Especificações do sistema
